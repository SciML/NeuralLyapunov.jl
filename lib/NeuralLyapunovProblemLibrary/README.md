NeuralLyapunovProblemLibrary.jl is a component of the [NeuralLyapunov.jl](https://github.com/SciML/NeuralLyapunov.jl) monorepo. It provides a library of benchmark dynamical-system models (pendula and quadrotors) as ModelingToolkit systems for testing and demonstrating neural Lyapunov function search.
While completely independent and usable on its own, users wanting the full neural Lyapunov function search suite should use [NeuralLyapunov.jl](https://github.com/SciML/NeuralLyapunov.jl).
